# АТЛАС ГОСТЯ

Демо **mobile-first web-приложения** для бутик-отеля — мобильный консьерж гостя, который открывается по ссылке без установки приложения.

Гость получает персональный доступ к расписанию дня, цифровой карте, консьерж-сервису, рекомендациям и привилегиям статуса.

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .
```

Затем откройте в браузере адрес с портом (например, `http://localhost:8080`).

## Структура

| Файл        | Назначение                          |
|-------------|-------------------------------------|
| `index.html`| Разметка экранов и навигации        |
| `styles.css`| Визуальный стиль soft premium       |
| `script.js` | Переключение вкладок и интерактив   |

Внешние библиотеки не используются.

## Вкладки

1. **Сегодня** — приветствие, номер, статус, расписание дня, вечерний план
2. **Карта гостя** — цифровая карта Private Guest с декоративным QR
3. **Консьерж** — бронь ужина, поздний выезд, трансфер, сообщение администратору
4. **Гид** — рекомендации маршрутов с мини-картой
5. **Привилегии** — список привилегий статуса Private Guest

## Для реального клиента

Это демонстрационный прототип. При внедрении для конкретного отеля можно заменить:

- название бренда и визуальный стиль (цвета, шрифты, логотип);
- услуги, расписание и тексты под реальный отель;
- QR-код на рабочий (генерация из ID гостя или PMS);
- ссылки на мессенджеры (WhatsApp, Telegram) вместо демо-сообщений;
- форму бронирования или интеграцию с CRM / PMS;
- данные гостя из backend по персональной ссылке.
